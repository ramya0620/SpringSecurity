# Spring Security Examples

This repository contains hands-on implementations and examples of Spring Security for Java web applications. It covers authentication, authorization, method-level security, and modern security standards like JWT, OAuth2, and OpenID Connect.

## Features & Learning Highlights

- Basics of **Spring Security** and how to integrate it into a Java web application.
- **Password Management** using PasswordEncoders, including hashing, encryption, and encoding techniques.
- **Security concepts:**
  - **Authentication vs Authorization**
  - **CSRF & CORS** and how to handle them
- **Filters** in Spring Security and creating custom filters for request processing.
- **JWT (JSON Web Tokens):**
  - How JWTs work in authentication & authorization
  - Token creation, verification, and usage
- **OAuth2 & OpenID Connect:**
  - Various OAuth2 grant flows
  - Role of OpenID Connect in authentication
  - Setting up an **Authorization Server** with Keycloak or Spring Authorization Server
- Applying **authorization rules** using roles and authorities in web applications.
- **Method-level security** in web and non-web applications.
- **Social login integration** into Spring applications.

## Project Structure

- `sec12MethodLevelSecurity/` → Examples of method-level security
- `sec14springsecOAUTH2/` → Examples of OAuth2/OpenID Connect & JWT
- `parivarbankui/` → Frontend Angular UI for testing security features
